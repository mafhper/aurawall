# Configuração e Baselines de Performance (performance/)

## Propósito

Este diretório armazena configurações e dados de baseline para as ferramentas de análise de performance do projeto. Ele é fundamental para o Performance Gate, que compara os resultados de auditorias atuais com metas e históricos.

## Conteúdo

### Subdiretórios
- `baseline/`: Contém relatórios Lighthouse de baselines de performance para diferentes "form factors" (mobile/desktop).

### Arquivos
- `thresholds.json`: Define os limites de "fail" e "warn" para as métricas de performance (LCP, TBT, CLS, etc.) para mobile e desktop.
