# Baselines de Performance (performance/baseline/)

## Propósito

Este diretório contém relatórios de performance do Lighthouse que servem como "baselines" (linhas de base) para comparação. Esses relatórios representam um estado de performance aceitável ou ideal, e são usados para detectar regressões em auditorias futuras.

## Conteúdo

### Arquivos
- `navigation.desktop.json`: Relatório de baseline para auditorias de navegação em desktop.
- `navigation.mobile.json`: Relatório de baseline para auditorias de navegação em mobile.
