# Assets Públicos (public/)

## Propósito

Este diretório contém todos os assets estáticos que são servidos diretamente pelo servidor web da aplicação principal. Estes arquivos são acessíveis publicamente e incluem ícones, favicons, imagens de background e o manifesto do PWA.

## Conteúdo

### Subdiretórios
- `favicon/`: Contém versões alternativas ou específicas de favicons.

### Arquivos
- `apple-touch-icon-dark.png`: Ícone para dispositivos Apple em modo escuro.
- `apple-touch-icon.png`: Ícone para dispositivos Apple.
- `bg-boreal.svg`, `bg-chroma.svg`, etc.: Arquivos SVG de background estáticos gerados pelos scripts.
- `favicon-16x16-dark.png`, `favicon-16x16.png`, etc.: Favicons em várias resoluções e modos (claro/escuro).
- `favicon-dark.ico`, `favicon.ico`: Favicons no formato ICO.
- `favicon-dark.svg`, `favicon.svg`: Favicons no formato SVG.
- `header-animation.svg`: SVG para animações no cabeçalho.
- `hero-mobile.webp`: Imagem otimizada para o hero em dispositivos móveis.
- `icon-light.svg`: Ícone da aplicação em versão clara.
- `manifest.json`: Manifesto do Progressive Web App (PWA), contendo metadados para instalação do app.
- `mstile-150x150-dark.png`, `mstile-150x150.png`: Ícones para Live Tiles do Windows.
- `og-image-dark.jpg`, `og-image-dark.webp`, `og-image.jpg`, `og-image.webp`: Imagens para Open Graph (compartilhamento social) em versões clara e escura, otimizadas para a web.
- `pwa-192x192-dark.png`, `pwa-192x192.png`, etc.: Ícones para PWA em várias resoluções e modos, incluindo versões "maskable".
