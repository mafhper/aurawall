# Funções Utilitárias (src/utils/)

## Propósito

Este diretório contém funções auxiliares e utilitários genéricos que são usados em várias partes da aplicação para tarefas comuns, como manipulação de dados, cálculos matemáticos, conversões de formato, etc.

## Conteúdo

### Arquivos
- `blobUtils.ts`: Funções para gerar e manipular formas "blob" (formas orgânicas e arredondadas).
- `codeGenerators.ts`: Utilitários para gerar trechos de código (por exemplo, CSS ou SVG) a partir das configurações do wallpaper.
- `colorUtils.ts`: Funções para manipulação de cores, conversões entre formatos (HSL, Hex, RGB) e cálculos de harmonia de cores.
- `compactUrlEncoder.ts`: Implementa um esquema de codificação compacto para salvar e carregar configurações de wallpaper via URL.
- `engineUtils.ts`: Funções utilitárias específicas para o sistema de engines de geração de wallpaper.
- `urlUtils.ts`: Utilitários para manipular parâmetros de URL, como codificar e decodificar configurações para compartilhamento.
