# Custom React Hooks (src/hooks/)

## Propósito

Este diretório contém hooks personalizados do React, projetados para encapsular lógicas de estado e comportamento reutilizáveis, promovendo uma arquitetura mais limpa e modular.

## Conteúdo

### Arquivos
- `useFavorites.ts`: Hook para gerenciar a lista de wallpapers favoritos do usuário, incluindo adicionar, remover e verificar favoritos.
- `useHistory.ts`: Hook para implementar funcionalidades de "desfazer" (undo) e "refazer" (redo) para as configurações do wallpaper, mantendo um histórico de estados.
