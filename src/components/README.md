# Componentes React (src/components/)

## Propósito

Este diretório abriga os componentes React que são a base da interface de usuário da aplicação principal. Cada componente é projetado para ser reutilizável e encapsular uma parte específica da UI e sua lógica associada.

## Conteúdo

### Arquivos
- `CodeExportModal.tsx`: Modal para visualização e exportação do código SVG gerado.
- `ColorPaletteExtractor.tsx`: Ferramenta para extrair e exibir a paleta de cores de um wallpaper gerado.
- `Controls.tsx`: O painel de controle principal da aplicação, onde o usuário interage com as configurações do wallpaper.
- `EngineGallery.tsx`: Componente para exibir uma galeria de motores de geração.
- `PreferencesMenu.tsx`: Menu de preferências do usuário (formato de exportação, prefixo de nome de arquivo, etc.).
- `WallpaperRenderer.tsx`: O componente central responsável por renderizar o wallpaper (SVG) com base na `WallpaperConfig`.
